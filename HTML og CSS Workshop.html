HTML og CSS workshop

I dette emne blev vi delt op i grupper på to. Vi fik hver uddelt et emne inden for HTML eller CSS som vi skulle undersøge, og derefter arbejde med så vi til sidst kunne præsentere hver vores emne overfor hinanden.

Opgave

Min gruppe fik emnet CSS selectors. Vi har vidst eksempler af de forskellige selectors og hvordan de bruges i både HTML og CSS kodning.

Det endte med, at vi valgte, at vise eksempler på en hjemmeside
